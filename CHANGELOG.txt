# Changelog 

## macchina v4.0, codename "Copper"

- macchina now looks for themes in `$XDG_CONFIG_HOME/macchina/themes`, the previous data path is deprecated. (#172)
- Many command-line flags have been deprecated in favor of being moved to/present in theme files. (#172)
- Themes are now written in TOML rather than JSON. (#172)
